# Paired Voting Website

Welcome to the Paired Voting Website project! This is a [Next.js](https://nextjs.org) application.

## Prerequisites

Before you begin, ensure you have the following installed on your computer:
- [Node.js](https://nodejs.org/) (version 18.17 or higher)
- [Git](https://git-scm.com/)

## Getting Started

Follow these step-by-step instructions to get the project running on your local machine:

### 1. Clone the Repository
First, you need to download the code to your computer. Open your terminal and run:

```bash
git clone https://github.com/christianpaglinawan/Paired-Voting-Website.git
```

### 2. Navigate to the Project Folder
Once cloned, move into the downloaded folder, and then specifically into the Next.js application folder (`my-app`):

```bash
cd Paired-Voting-Website/my-app
```

### 3. Install Dependencies
You need to install the required packages (like Next.js and React) for the project to run. Still inside the `my-app` folder, run:

```bash
npm install
```

### 4. Run the Development Server
After the installation finishes, you can start the development server:

```bash
npm run dev
```

### 5. View the Website
Open your web browser and go to [http://localhost:3000](http://localhost:3000). You should now see the website running locally!

## Making Changes
- You can start editing the page by modifying `app/page.tsx`.
- The page will automatically update in the browser as you edit and save the file.
